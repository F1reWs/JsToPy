# JsToPy 0.1.3
A library that add to python some js functions.

## Installation
`pip3 install JsToPy` or `pip3 install --user JsToPy`

## Why JsToPy?
JsToPy makes JavaScript code using in Python.

JS code:
```js
var this = require("a-lib");

console.log(nums);
```

Using JsToPy:
```py
import JsToPy

this = require("a-lib");

console.log(nums);
```
